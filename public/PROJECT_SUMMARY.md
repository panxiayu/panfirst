## 🚀 部署方式

### 方式 1：本地运行

```bash
cd D:\exam\server
npm start
```

访问：
- 首页: http://localhost:3000/
- 考试: http://localhost:3000/exam-list.html
- 投票: http://localhost:3000/voting-list.html
- 报餐: http://localhost:3000/meal-list.html

### 方式 2：Docker 部署

```bash
cd D:\exam\server
docker-compose up -d
```

访问：
- http://192.168.110.22:3000/

### 方式 3：Linux 部署

```bash
ssh -p 2222 openclaw@192.168.110.22
cd /home/openclaw/apps/server
docker-compose up -d
```
