## 📝 使用说明

### 首次使用

1. **启动后端服务**
```bash
cd D:\exam\server
npm start
```

2. **打开投票系统**
```
http://localhost:3000/voting-list.html
```

3. **登录**
- 输入任意 openid（用于测试）
- 系统会自动创建用户

4. **创建投票**
- 点击"创建投票"按钮
- 填写投票信息
- 添加选项
- 提交

5. **参与投票**
- 点击投票卡片
- 选择选项
- 提交投票

### 报餐系统

1. **打开报餐系统**
```
http://localhost:3000/meal-list.html
```

2. **创建报餐**
- 点击"创建报餐"按钮
- 填写报餐信息
- 添加菜单
- 提交

3. **参与报餐**
- 点击报餐卡片
- 选择菜单
- 调整数量
- 输入备注
- 提交
