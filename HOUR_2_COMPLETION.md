## 📝 API 集成

### 已集成的 API 端点

**人员管理**:
- GET /api/staff - 获取员工列表
- GET /api/staff/:id - 获取员工详情
- POST /api/staff - 添加员工
- DELETE /api/staff/:id - 删除员工

**工作任务**:
- GET /api/task - 获取任务列表
- GET /api/task/:id - 获取任务详情
- POST /api/task - 创建任务
- POST /api/task/:id/complete - 完成任务
- DELETE /api/task/:id - 删除任务

**系统设置**:
- GET /api/settings - 获取所有设置
- PUT /api/settings - 更新设置
